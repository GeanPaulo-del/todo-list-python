# 📋 Sistema de Lista de Tarefas em Python

Sistema simples de lista de tarefas feito em Python no terminal.

## 🚀 Funcionalidades

- Adicionar tarefas
- Listar tarefas
- Marcar tarefa como concluída
- Remover tarefa
- Armazenamento em arquivo JSON

## 🧠 Tecnologias

- Python
- JSON

## ▶️ Como executar

```bash
python main.py
📁 Estrutura
main.py → sistema principal
tarefas.json → armazenamento das tarefas
🎯 Objetivo
Projeto criado para praticar lógica de programação e manipulação de dados em Python.
